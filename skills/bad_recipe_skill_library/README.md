# 难喝奶茶配方 Skill 库

这个目录用于存放反面配方样本，帮助 AiNaiCha 项目在生成新品时识别并避开风味冲突、口感失衡、成本不合理、出杯效率差和表达混乱的奶茶方案。

分类指标与正向配方库保持一致：季节、价格带、成本上限、出杯时间、目标人群、甜度倾向、温度形态。

说明：本目录中的配方是人为构造的负面样本，不应作为真实可售配方。每条记录包含配方名、反面 ID、问题类型、杯量、糖度、估算成本、原料克重和主要问题。
